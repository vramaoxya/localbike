{% docs mrt_order_shipped_montly %}

This model provides an aggregated view of the orders sent. It enriches the order data with the following metrics:
- **nb_order_shipped**: The quantity of orders shipped.
- **total_sales**: The cost of orders shipped.
- **Order Informations**: Enriches the product with user-specific data, such as store name.

It provides a comprehensive view the aborted order, allowing for easy analysis of order performance.
