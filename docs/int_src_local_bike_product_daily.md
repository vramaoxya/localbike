This model provides an aggregated view of product items. It enriches the products data with the following metrics:
- **Total Order Amount**: The sum of all order items for each product.
- **Total Items**: The quantity of items in the store.
- **Product Informations**: Enriches the product with user-specific data, such as brand and category.

It provides a comprehensive view of each product, allowing for easy analysis of product performance.
